Introduction

HTML, or Hypertext Markup Language, is a markup language for the web that defines the structure of web pages.

With HTML code you can make any kind of webpage your mind can imagine!

    • Doctype Declaration and Meta Elements

    To get started you need to write whats known as a declaration. This tells the browser what type of code it will be reading. Since we will be using HTML code we will write it like this

    <!DOCTYPE html>

    Next we need to set the character set that the browser will read from, this will be utf-8. To set this we use the meta element.

    <meta charset="utf-8">

    The final thing in this section will be setting the viewport. This ensures that the page will appear the same on mobile devices.

    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    Title Element

    The title element will tell the browser what the name of the page is

    <title>Page Title</title>

    This will show on tabs when you navigate to the page

    Main Element

    The main element is where all of you code is held for the entire page

    <Main> <Main>
    Header Element

    The header element serves many purposes you will have one at the top of your page. It can also be used to make headers for text areas of your page

    <header> <header>
